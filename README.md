# BlueGuard Website

Premium animated landing page for **BlueGuard Authenticator** with:
- strong scroll/motion effects
- multilingual UI (EN, JA, RU, UK, PT)
- legal pages for App Store review

## Files
- `index.html` - main marketing website
- `styles.css` - visual system and animations
- `script.js` - i18n, scroll effects, interaction logic
- `terms.html` - Terms of Use
- `privacy.html` - Privacy Policy
- `service-agreement.html` - Service Agreement

## Assets
Put your media into `/assets` with these exact names:
- `logo.png`
- `hero-main.png`
- `screen-authenticator.png`
- `screen-passwords.png`
- `screen-notes.png`
- `screen-onboarding.png`

See `/assets/README.md`.

## Run
Open `index.html` in browser.
